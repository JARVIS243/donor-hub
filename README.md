# 🩸 Donor Hub

A full-stack Blood Bank Management System built using Django that connects donors with recipients and helps manage emergency blood requests efficiently.

---

## 🚀 Features

- 🔐 User Authentication (Signup / Login / Logout)
- 🩸 Donor Registration & Management
- 🔍 Search Donors by Blood Group
- 🚑 Emergency Blood Request System
- 📊 Blood Availability Dashboard
- 🧑‍💻 User-specific Donor Management (My Donors)
- ⚙️ Admin Panel for full control
- 📱 WhatsApp Contact Integration

---

## 🛠️ Tech Stack

- **Backend:** Python (Django)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
- **Tools:** Git, GitHub

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/donor-hub.git
cd donor-hub
````

---

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Run migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

---

### 5. Create superuser

```bash
python manage.py createsuperuser
```

---

### 6. Run server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
